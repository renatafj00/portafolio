    • tipografía
    2022

Este proyecto consistió en diseñar una identidad gráficamy un sistema de acomodo tipográfico que se replicaría mes con mes en una cartelera para el museo MALBA.
Se hizo una exploración de formato para establecer la lectura y el entendimiento tipográfico en papel blanco de 43x56cm.
Colectivamente se generó material para la identidad gráfica y se llevó a cabo el tratamiento fotográfico.
